Captronic Former le monde de l'embarqué à TDD.

Captronic - Février 2023 - Bordeaux
TDD · Java · BDD · Test · Code Legacy · Golden Master · Defect Testing

Formation TDD sur 3 jours auprès d'un public de développeurs, majoritairement dans l'embarqué. TDD a été abordé sur des projets neufs, mais le sujet du code Legacy a aussi été debroussaillé.

La formation est partie du B-A-BA des tests pour cheminer jusqu'au plus complexe : introduire des tests sur un code qui n'en a jamais connu. Nous avons suivi un fil rouge afin d'apprendre le recueil du besoin piloté par les tests.
