Legacy Code · Refactoring · Tests · Anticorruption Layer · Defect Testing · Golden Master

Cette formation vise à enseigner aux développeurs les techniques pour poursuivre un projet ayant une forte complexité accidentelle et un manque criant de tests, aussi appelé Code Legacy.

Des techniques comme les Golden Master, les Seam ou encore le Defect Testing sont transmises, accompagnées d'un bagage théorique sur les stratégies de résorption du Code Legacy.
